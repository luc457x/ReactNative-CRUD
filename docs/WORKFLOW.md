# Diretriz Universal de Workflow (Obrigatória)

1. **Antes de iniciar uma tarefa:** Sempre que for solicitado o início de uma tarefa, verifique no arquivo `AGENTS.md` (na raiz) qual agente é responsável pela Fase em que a tarefa se encontra, e adote as diretrizes/persona dele para a execução.
2. **Gerenciamento de Conflitos:** Se você cometer um erro ou precisar alterar um fluxo já definido no `docs/TASKS.md`, **NÃO** modifique o arquivo diretamente. Crie uma nova tarefa de "Correção" ou "Refatoração" com um título claro (ex: "[FIX] Ajuste no fluxo de Login") e marque a tarefa antiga como **Bloqueada** ou **Resolvida** com uma nota explicativa.
3. **Antes de concluir** qualquer tarefa e atualizar o `TASKS.md`, você **DEVE OBRIGATORIAMENTE** seguir o Processo de Validação detalhado na seção 2 do **[`docs/HARNESS.md`](docs/HARNESS.md)**.
